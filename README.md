# fundamentos-nodejs

Projeto API RESTful com Node.js, com o propósito de aprofundar meus conhecimentos nos fundamentos da tecnologia, sem depender de frameworks ou bibliotecas externas.

Durante o projeto, explorei os módulos internos do Node.js, tais como HTTP, Crypto e File System, e aprofundei-me nos fundamentos do protocolo HTTP, como requests, responses, headers, status code, route e query parameters.

Além disso, busquei compreender o uso das Streams no Node.js, aplicando-as para a realização de operações assíncronas e parciais no back-end.
